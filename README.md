# 🚗 Uber Ride Price Prediction (Machine Learning Project)

## 📌 Project Overview
The Uber Ride Price Prediction project uses Machine Learning to estimate the fare of a ride based on  location, travtravel_time_min, traffic_level. It helps users predict ride prices accurately and supports better cost planning.

![ml](https://github.com/rushikeshwalode06-cod/Uber-Ride-Price-Prediction/blob/main/Uber%20Ride%20Price%20Prediction.png?raw=true)

## 🎯 Objective
1. Predict ride fare using ML
2. Improve pricing accuracy
3. Assist users in estimating ride cost

## 📂 Dataset Information
Features used in dataset:
1. distance_km
2. travel_time_min
3. traffic_level
4. base_fare
